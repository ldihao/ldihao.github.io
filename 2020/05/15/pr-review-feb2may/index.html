Paper Review - Review from Feb. to May.

2020-05-15
Paper review
649 words 2 mins read times read

  • Attention map

总结一下从2月到5月看的9篇文章。

  • Segmentation
    • AG-Net: 使用注意力机制+线性模型将高层次的特征作为富含内容的信息与低层次的高分辨率的特征相结合，得到高分辨率同时含有内容信息的分割图像。
    • Uniqueness-Driven Saliency Analysis: 从特征提取角度，根据损伤部位具有独特性的特点，对每个像素点的颜色和位置方面考虑，得到各自的独特性，由此分割出眼部损伤部位。
    • ASD-Net: 使用半监督学习和对抗学习的方法，将没有像素级别标记的数据加入训练，提高分割模型泛化能力。
  • Classification
    • PSE-Net: 牛皮癣检测，分别对深度学习模型每个层次的特征图进行牛皮癣位置和严重程度的预测，使用分割的损失函数使模型注意力集中在病理位置，提高疾病诊断准确性。
    • AG-CNN: 利用专家注意力图的青光眼检测
    • AF-Net: 使用眼部损伤部位来诊断糖尿病性视网膜突变程度，针对损伤分割图不够精确的问题，使用采样的方法减少噪声的影响
    • CIA-Net: 正负样本特征图的差异度作为 attention map 的基础，结合设计的对齐模块来使不同样本对齐，以方便 location-wise 相减，达到了分类结果准确率提高。
  • Seg + Clf
    • CL-Net: 使用半监督的方法训练眼部损伤图像的分割模型，同时结合 attention map 分类诊断 DR 严重程度。
  • 其他
    • Explanation By Progressive Exaggeration

模型 得到方法 处理方式 如何融合
AG-Net 高层次富含内容的信息 线性模型（低分辨率） 线性模型 upsampling 后作用在低层次高分辨率特征图上
ASD-Net 判别器的输出 不处理 与生成模型的结果相乘
:–: :–: :–: :–:
PSE-Net 与GT拟合 不处理 与score map点乘
AG-CNN 专家的注意力图 训练输入为点乘处理后图像的分类网络，guided-BP算法，得到注意力图 与input点乘
AF-Net 特征图 attention network (conv3x3+conv1x1) 与特征图点乘
CIA-Net 正负样本特征图相减 normalization (sum=H*W) 与特征图相乘
CL-Net 切割模型 CNN 低层次（H*W）+高层次（1*1）点乘
