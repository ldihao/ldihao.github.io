Paper review - [2019CVPR] Attention Based Glaucoma Detection: A Large-scale Database and CNN Model

该论文通过对专家诊断青光眼时对眼底图像区域的记录，将专家注意力图像作为特征进行预测，将得到的注意力图与全局图像点乘，再进入CNN得到最后的概率。

General

  • use case: 青光眼诊断
  • 技术上：
    • 由于普通的CNN会使用许多冗余信息，采取 attention 机制，将网络的注意力放在视杯区域
    • 使用 multi-scale block 提取特征
    • 使用 guided-BP 得到 visualization map
  • 结果：89.7% -> 95.3%

Introduction

Main contribution:

  • 创建了一个 LAG database （5824 fundus images, with positive glaucoma (2,392) and negative glaucoma (3,432))
  • 提出 attention map
  • 建立 AG-CNN，which visualizes the CNN feature maps for locating pathological area and then classifies binary glaucoma

数据集

5,824 labeled fundus images with 2,392 positive and 3,432 negative glaucoma samples.

Experiment to capture the attention regions of the ophthalmologists in glaucoma diagnosis.

基于眼科专家诊断时眼睛注视区域的变化顺序，通过高斯模糊，生成一个 attention map。该试验共有4个专家参加，专家之间（one vs others）的皮尔森相关系数均大于0.55，说明有相关性。同时发现，ROI占总图片比例小于20%，说明 attention 机制的可行性。

Method

Attention Prediction subnet

基于 ResNet50，使用 feature normalization block 将每一层特征图整合为128*28*28，再使用 deconvolutional module 生成 1*112*112 的 attention map。此结果的 GT 为 Database 中的 attention map.

Pathological Area Localization Subnet

使用四个 multi-scale building block 作为 backbone，每一层的 feature map 都乘上上一个 subnet 得到的 attention map，最后用两个全连接网络得到青光眼概率。

使用 guided BP 算法，得到在 input fundus image 上的影响区域，此为需要得到的 Pathological Area 病理区域。

Glaucoma Classification Subnet

与上一个 subnet 相同的结构，但每一层乘上的是与 Pathological Area 相关的 mask。

Loss

  • Attention Loss: KL divergence
  • Visualization Loss: 交叉熵
  • Classification Loss: 交叉熵
  • Overall: weighted

总结

感觉这篇文章最大的贡献点还是引入了专家知识（attention map 的建立），第二部分使用网络自身学习新的病理区域提高较少（2%）。

医学图像分析最重要的还是医学知识的引入。
