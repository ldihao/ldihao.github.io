Paper review - [2018MICCAI] A Framework for Identifying Diabetic Retinopathy Based on Anti-noise Detection and Attention-Based Fusion

General

  • 场景：Diabetic Retinopathy 多分类
  • 技术上：
    • lesion 对 DR 识别有帮助，但许多 lesion 难以找到，存在 unannotated samples 造成 noise
    • 使用 center-sample detector 减少这些噪声的影响
    • 使用 Attention fusion network 综合分割和fundus信息
  • 在两个公开数据集 EyePACS，Messidor和一个私有数据集上效果好

Method

总体分为分割和分类两部分。

分割网络在 SSD 基础上增加 center-sample detector。通过降低 noise 的采样概率，减轻 unannotated sample 对结果的影响。

AFN 使用 ResNet18 作为 feature extractor，提取出的 feature map 再用简单CNN网络得到 weight，最后乘上原有特征再 concat，得到最终 feature map。

Center-Sample Detector

SSD 网络：SSD采用CNN来直接进行检测，SSD提取了不同尺度的特征图来做检测。使用 multi-scale 作为输入，大尺度特征图（较靠前的特征图）可以用来检测小物体，而小尺度特征图（较靠后的特征图）用来检测大物体；另外，SSD采用了不同尺度和长宽比的先验框（Prior boxes, Default boxes，在Faster R-CNN中叫做锚，Anchors）。

本文在 detect header 之前加入了 center sample detector 来控制进入下一步的样本。

在训练时，需要先 pretrain SSD 部分，再加上 sample clustering。

Sample Clustering

使用 cluster 方法，将不同 deep feature 聚类，认为虽然没有标记但聚为一类的样本为 lesion 即 unannotated sample（nosie）。

Noisy Sample Mining

使用 down-weighting 来减少这部分样本的影响。具体使用 threshold rank 来处理这些样本的采样概率。

$$P\left(u_{i j}\right)=\left\{\begin{array}{ll} 0 & 0 \lt r_{i j} \lt t_{l} \\ \left(\frac{r_{i j}-t_{l}}{t_{u}-t_{l}}\right)^{\gamma} & t_{l} \leq r_{i j} \lt t_{u} \\ 1 & r_{i j} \geq t_{u} \end{array}\right.$$

Attention Fusion Network

The attention network consists of a 3 × 3 Conv, a ReLU, a dropout, a 1 × 1 Conv, and a Sigmoid layer.

The final output is produced by performing a softmax operation on f(i,j,c) to get the probabilities of all grades.
