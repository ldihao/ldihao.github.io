由于医学图像分割中，有像素级别标记的数据集较少，本文使用半监督学习和对抗学习的方法，将没有像素级别标记的数据加入训练，提高模型泛化能力。

General

  • 场景：骨盆器官分割
  • 挑战：pixel-level 标记的数据过少
  • 贡献：使用 adversarial、semi-supervise、sample attention 的方法，降低数据过少的影响
    • 其中 semi-supervised 方法，使用了 region based attention，将没有标记的图像加入训练
  • 数据集：
    • pelvic dataset (private, 50 images + 20 non-labeled images)
    • PROMISE12-challenge dataset: competitive results（但是没有具体数据）

Method

Notation

  • Labeled input image: $\mathbf{X} \in R^{H \times W \times T}$ where T: time
  • Corresponding ground-truth label map: $\mathbf{Y} \in Z^{H \times W \times T}$
  • Its one-hot format: $\mathbf{P} \in R^{H \times W \times T \times C}$

Segmentation network with sample attention

model: simplified VNet (internal pool-conv-deconv layers are removed, and thus is denoted as SVNet)

Multi-class Dice Loss

$$L_{\text {Dice}}\left(\mathbf{X}, \mathbf{P} ; \theta_{\mathbf{s}}\right)=1-2 \frac{\sum_{c=1}^{C} \pi_{c} \sum_{h=1}^{H} \sum_{w=1}^{W} \sum_{t=1}^{T} P_{h, w, t, c} \widehat{P}_{h, w, t, c}}{\sum_{c=1}^{C} \pi_{c} \sum_{h=1}^{H} \sum_{w=1}^{W} \sum_{t=1}^{T} P_{h, w, t, c}+\widehat{P}_{h, w, t, c}}$$

where $\pi_c$ is class balancing weight

Multi-class Dice Loss with Sample Attention

Inspired by focal loss:

缓解被 easy samples 掌控的问题。

$$L_{A t t D i c e}\left(\mathbf{X}, \mathbf{P} ; \theta_{\mathbf{s}}\right)=(1-d s c)^{\beta}\left(1-2 \frac{\sum_{c=1}^{C} \pi_{c} \sum_{h=1}^{H} \sum_{w=1}^{W} \sum_{t=1}^{T} P_{h, w, t, c} \widehat{P}_{h, w, t, c}}{\sum_{c=1}^{C} \pi_{c} \sum_{h=1}^{H} \sum_{w=1}^{W} \sum_{t=1}^{T} P_{h, w, t, c}+\widehat{P}_{h, w, t, c}}\right)$$

每一次 iteration 重新计算 $\textit{dsc}$。

Confidence network (Discriminator)

Adversarial learning

Adversarial Loss of the Confidence Network

$$L_{D}\left(\mathbf{X}, \mathbf{P} ; \theta_{\mathbf{d}}\right)=L_{B C E}\left(D\left(\mathbf{P}, \theta_{\mathbf{d}}\right), \mathbf{1}\right)+L_{B C E}\left(D\left(S(\mathbf{X}), \theta_{\mathbf{d}}\right), \mathbf{0}\right)$$

  • GT mask -> true (1)
  • pseudo mask -> false (0)

Adversarial Loss of the Segmentation Network

$$L_{A D V}\left(\mathbf{O}, \theta_{\mathbf{s}}\right)=L_{B C E}\left(D\left(S\left(\mathbf{O} ; \theta_{\mathbf{s}}\right)\right), \mathbf{1}\right)$$

让 segmentation model 计算出的结果尽量为真。

Region-Attention Based Semi-supervised Learning

Discriminator (i.e., confidence network):

  • segmentation model,
  • 提供 pixel-level local confidence map
  • $\mathbf{M}=D(\widehat{\mathbf{P}})$

Confident region: $[\mathbf{M} \le \gamma]$

Semi-supervised loss: $$L_{\text {semi}}\left(\mathbf{U}, \theta_{\mathbf{s}}\right)=1-2 \frac{\sum_{c=1}^{C} \pi_{c} \sum_{h=1}^{H} \sum_{w=1}^{W} \sum_{t=1}^{T}[\mathbf{M}>\gamma]_{h, w, t} \bar{P}_{h, w, t, c} \widehat{P}_{h, w, t, c}}{\sum_{c=1}^{C} \pi_{c} \sum_{h=1}^{H} \sum_{w=1}^{W} \sum_{h, w, t, c}^{T} \sum_{h=1}^{T}[\mathbf{M}>\gamma]_{h, w, t}\left(\bar{P}_{h, w, t, c}+\widehat{P}_{h, w, t, c}\right)}$$

where $\bar{P}$ is the one-hot encoding of $\widehat{Y}$, and $\widehat{Y} = \arg \max{(\widehat{P})}$. $[]$ is the indicator function. $\bar{P}$ and $[\mathbf{M} \le \gamma]$ 每次 iteration 后重新计算。

Experiment

  • input size for segmentation model: $64\times64\times16$
  • image size in dataset: $256\times256\times(120-176)$
