DNN

Problem:

$D=\left\{\left(\mathbf{x}^{(1)}, y^{(1)}\right),\left(\mathbf{x}^{(2)}, y^{(2)}\right), \ldots\right\}$

$\underset{\omega}{\mathrm{argmin}} \quad {\mathbf{y}-f_{\omega}(\mathbf{x})}$

  • sparsity: the number of zero elements in $\mathbf{x}$ » n
  • #instance 通常非常大，需要分布式计算
  • order-2 interaction 非常重要

FM - Factorization Machines

一种分类器，与 SVM、factorization models 相比：

  • model all interactions between variables using factorized parameters (where SVM fails when facing sparse data)
  • applicable to standard data ($\in \mathbb{R}^n$)
  • be calculated in linear time

FM model

  • degree = 2 时：

    $\hat{y}(\mathbf{x}):=w_{0}+\sum_{i=1}^{n} w_{i} x_{i}+\sum_{i=1}^{n} \sum_{j=i+1}^{n}\left\langle\mathbf{v}_{i}, \mathbf{v}_{j}\right\rangle x_{i} x_{j}$

    $\text{where: } w_{0} \in \mathbb{R}, \quad \mathbf{w} \in \mathbb{R}^{n}, \quad \mathbf{V} \in \mathbb{R}^{n \times k} ,\quad \mathbf{x} \in \mathbb{R}^n$

    • 此时，FM 提取了所有的 single & pairwise interactions

Pairwise interactions

  • SVM with polynomial kernel：$\hat{y}(\mathbf{x}):=w_{0}+\sum_{i=1}^{n} w_{i} x_{i}+\sum_{i=1}^{n} \sum_{j=i+1}^{n}\mathbf{u}_{i,j} x_{i} x_{j}$

    $\text{where: } \mathbf{U} \in \mathbb{R}^{n\times n}$

  • 将 $\mathbf{U}$ 替换为 $\mathbf{V} \cdot \mathbf{V}^t$，根据定理，如果k足够大，$\mathbf{V} \cdot \mathbf{V}^t$ 可以表示任意正定矩阵。但为了模型泛化能力，一般 $k$ 较小。

  • 且由于不同词条（$\mathbf{x}^{(j)}$）的相同特征（$\mathbf{x}_i^{(j)}$）共享了同一个权重参数（$\mathbf{v}_i$）,在 sparse 特征下（dense data 下是否效果不会这么明显？），这个权重综合了所有词条的特征关系。

FFM - Field-aware Factorization Machine

由于使用了 one-hot 编码再 concat 特征，不同类型特征之间的差异性被忽视。FFM 将 features 分到不同的 fields 中，同一 feature 对不同 field 中的 feature 使用不同 $\mathbf{v}_{j,f}$。

$$ \phi_{\mathrm{FFM}}(\boldsymbol{w}, \boldsymbol{x})=\sum_{j_{1}=1}^{n} \sum_{j_{2}=j_{1}+1}^{n}\left(\boldsymbol{w}_{j_{1}, f_{2}} \cdot \boldsymbol{w}_{j_{2}, f_{1}}\right) x_{j_{1}} x_{j_{2}} $$

由于增加 field 信息，#variable = $nfk$， complexity = $O(n^2k)$， 且 $k_{\mathrm{FFM}} \ll k_{\mathrm{FM}} (40 \to 4)$

Deep and Wide

对于一个推荐系统：

  • Memorization: 从历史数据中找到 user 与 item 间的相关性 $\to$ wide model
  • Generalization: 发现在历史数据中很少或者没有出现的新的特征组合 $\to$ deep model

Deep and Wide: 两种模型并行计算，统一计算 loss，形成一个 end-to-end 的模型。

Prediction:

$$P(Y=1 \mid \mathbf{x})=\sigma\left(\mathbf{w}_{\text {wide}}^{T}[\mathbf{x}, \phi(\mathbf{x})]+\mathbf{w}_{\text {deep}}^{T} a^{\left(l_{f}\right)}+b\right)$$

  • embedding 的具体操作？（tf doc 里没写清楚？好像可以随着 training 变化）
  • wide 部分没有使用 $\mathbf{x}$？只使用了 cross product？

DeepFM

融合 FM（low-order）和 DNN（high-order），不需要手动 cross-product。

DNN 第一层直接使用 FM 中的 latent vector $\mathbf{V}$ 作为 dense embedding 参数。

比较了其他 DNN 的结构：FNN、PNN、Wide & Deep。

Experiment 中说 offline 一点点的提高（0.275%）可能导致 online gain 的巨大提高（3.9%）：可能由于 gain 随 auc 的变化并不是线性的。

GCNN - Graph Convolutional Network

Previous 问题（semi-supervised node classification task）定义：

$$\mathcal{L}=\mathcal{L}_{0}+\lambda \mathcal{L}_{\mathrm{reg}}, \quad \text { with } \quad \mathcal{L}_{\mathrm{reg}}=\sum A_{i j}\left\|f\left(X_{i}\right)-f\left(X_{j}\right)\right\|^{2}=f(X)^{\top} \Delta f(X)$$

GCNN 将 graph structure 加入 model 的表达中，将 loss 简化为 $\mathcal{L}_{0}$。

卷积层定义

$$ H^{(l+1)}=\sigma\left(\tilde{D}^{-\frac{1}{2}} \tilde{A} \tilde{D}^{-\frac{1}{2}} H^{(l)} W^{(l)}\right) \text{ where: } \tilde{A}=A+I_{N}, \tilde{D}_{i i}=\sum_{j} \tilde{A}_{i j} $$

with $W^{(l)} \in \mathbb{R}^{C \times F}, X \in \mathbb{R}^{N \times C}$

思想：将 nodes 转换到由 graph structure 定义的 Fourier domain 中，使用 filter 过滤信息，再转换回原来空间中。

问题

  • does not naturally support edge features and is limited to undirected graphs
    • possible to handle both directed edges and edge features by representing the original directed graph as an undirected bipartite graph with additional nodes that represent edges in the original graph

GraphSAGE

GCNN 的问题：

  • Transductive：不能很好迁移到结构不同的图上
  • 矩阵计算量大

GraphSAGE：

  • spectral domain -> spatial domain
    • 重新定义卷积操作（aggregator functions (for neighbors) + concatenate）
$$\mathbf{h}_{\mathcal{N}(v)}^{k} \leftarrow \operatorname{AGGREGATE}_{k}\left(\left\{\mathbf{h}_{u}^{k-1}, \forall u \in \mathcal{N}(v)\right\}\right) \\ \mathbf{h}_{v}^{k} \leftarrow \sigma\left(\mathbf{W}^{k} \cdot \operatorname{CONCAT}\left(\mathbf{h}_{v}^{k-1}, \mathbf{h}_{\mathcal{N}(v)}^{k}\right)\right)$$

问题：

  • 每个 node 最后的 feature 最多只能综合距离最远为 K 的 node；link prediction 时最远 2K 的 node 可能有相同邻居？

Loss function

unsupervised learning:

$$ J_{\mathcal{G}}\left(\mathbf{z}_{u}\right)=-\log \left(\sigma\left(\mathbf{z}_{u}^{\top} \mathbf{z}_{v}\right)\right)-Q \cdot \mathbb{E}_{v_{n} \sim P_{n}(v)} \log \left(\sigma\left(-\mathbf{z}_{u}^{\top} \mathbf{z}_{v_{n}}\right)\right) $$

融合

并不是仅仅单纯为了融合 DNN 与 RNN，而都是为了解决实际任务中的某种问题。

  • Hierarchical Representation Learning for Bipartite Graphs (Bi-HGNN)
    • embedding 时简单串型链接 Wide&Deep 和 GraphSAGE，使用 space decomposition 考虑 community-aware information，将 user feature 分解为 individual 和 community feature 再 fc。
  • Wide and Deep Graph Neural Networks with Distributed Online Learning (WD-GNN)
    • 为了解决 GNN 不能 online training，online learning 不能 distribute 的问题，融合线性模型和 GNN

Bi-HGNN

  • 场景
    • user item 推荐系统（二部图）
  • 挑战
    • 在 GNN 基础上使用 community-wise 信息
  • 贡献
    • 使用线性代数中 orthogonal base 分解的思想将 user feature 分解到 community feature space 和 individual user feature space 上
  • 数据集
    • MovieLens dataset
    • private e-commerce dataset

  • 首先使用 W&D embedding node feature，再使用 GraphSage 融合图的结构特征，得到 Full Embedding。
  • 对 user embedding 分解，初始随机生成 community-wise base $\{e_k^{S}, k\in [\![1,N ]\!] \}$ 且 $dim({e_k^{S}})<dim(e_u^{(F)})$，community-wise embedding 由 $e_u^{(F)}$ 和 $e_k^{S}$ 距离构成，individualized embedding 由分解得到。
  • 最后简单 concat 经过 mlp 得到最后结果。

问题：

  • 为什么 item feature embedding 时要使用 item-item edge？（using collaborative filtering algorithm）

Algorithm

$$ \mathbf{\Psi}(\mathbf{X} ; \mathbf{S}, \mathcal{A}, \mathcal{B})=\alpha_{\mathrm{D}} \mathbf{\Phi}(\mathbf{X} ; \mathbf{S}, \mathcal{A})+\alpha_{\mathrm{W}} \mathbf{B}(\mathbf{X} ; \mathbf{S}, \mathcal{B})+\beta $$

  • wide part - bank of linear graph filters (少了激活函数的 GNN)

$$ \mathbf{B}(\mathbf{X} ; \mathbf{S}, \mathcal{B})=\sum_{k=0}^{K} \mathbf{S}^{k} \mathbf{X} \mathbf{B}_{k} $$

  • deep part - GNN
$$\mathbf{\Phi}(\mathbf{X} ; \mathbf{S}, \mathcal{A})=\mathbf{X}_{L} \quad \text { with } \quad \mathbf{X}_{\ell}=\sigma\left(\sum_{k=0}^{K} \mathbf{S}^{k} \mathbf{X}_{\ell-1} \mathbf{A}_{\ell k}\right)$$

online training

  1. train the model - both wide and deep part
  2. test - online training wide part (convexe)
  • distribute online training
$$\mathcal{B}_{t+1}=\mathcal{B}_{t}-\gamma_{t} \nabla_{\mathcal{B}} J_{t}\left(\boldsymbol{\Psi}\left(\mathbf{X}_{t} ; \mathbf{S}_{t}, \mathcal{A}^{\dagger}, \mathcal{B}\right)\right)$$

$$ \Downarrow $$

$$\mathcal{B}_{i, t+1}=\frac{1}{\left|\mathcal{N}_{i}\right|+1}\left(\sum_{j: n_{j} \in \mathcal{N}_{i}} \mathcal{B}_{j, t}+\mathcal{B}_{i, t}\right)-\gamma_{t} \nabla_{\mathcal{B}} J_{i, t}\left(\mathbf{\Psi}\left(\mathbf{X}_{t} ; \mathbf{S}_{t}, \mathcal{A}^{\dagger}, \mathcal{B}_{i, t}\right)\right)$$

分解到每个 node 上分别计算 descent gradient，每轮计算后再 share。（有一定损失）
