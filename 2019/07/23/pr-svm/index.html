General

  • Title: A Novel Twin Support-Vector Machine With Pinball Loss
  • 贡献：提出了 Pin-TSVM，将 Pinball Loss 运用在 TSVM 上，降低了其对噪声的敏感性
  • https://ieeexplore.ieee.org/document/7381657

Introduction

  • Twin SVM
    • each hyperplane is closer to one class and as far as possible from the other
    • less computation
  • Pinball loss instead of hinge loss
    • reduce noise sensitivity

Related work

  • Loss function
    • Hinge loss: max{1-yf(x), 0}, y = ±1
    • Pinball loss: 1-yf(x) when 1-yf(x)≥0, -t(1-yf(x)) when 1-yf(x)<0
      • additional penalty on the correctly classified points
  • Twin Parametric-Margin Support-Vector Machine
    • 分为正反两组数据
    • 正 hyperplan 正确分辨 positive 数据的同时，远离 negative 数据，反之亦然
    • 最后两 hyperplan 相加为 decision function

NOVEL TWIN SUPPORT-VECTOR MACHINE WITH PINBALL LOSS

  • 将 TSVM 和 Pinball loss 相结合
    • Lagrangian function + KKT conditions
    • kernal trick still works

PROPERTIES OF THE Pin-TSVM

  • Noise Insensitivity
    • 由于 pinball loss 的加入，即使是判断正确的数据也会在 loss 中体现，于是 hyperplan 便会更偏向正例一边，所以对噪音敏感度降低
  • Between-Class Distance, Within-Class Scatter, and Misclassification Error
    • Pin-TSVM realizes the between- class distance maximization and makes the within-class scatter and the misclassification error minimization together.

COMPARISON WITH OTHER ALGORITHMS

  • Pin-TSVM Versus TPMSVM
    • 当 t 足够小时，pinball loss 降级为普通的 hinge loss
  • Pin-TSVM Versus Pin-SVM
    • faster
    • different t for two hyperplan: more adaptive than the Pin-SVM to unbalanced data

NUMERICAL EXPERIMENTS

  • Artificial Data Set
    • 人为增加噪音，展现出 Pinball-TSVM 抗噪能力强
  • UCI Data Sets With Noise
    • 与 SVM, TSVM 相比，有高有低
